## Recommended Free Image Sources

1. **Unsplash** (unsplash.com) — Best for portrait and lifestyle photos. Download full resolution, free for commercial use.
2. **Pexels** (pexels.com) — Good for activity/action shots. Free commercial use.
3. **Pixabay** (pixabay.com) — Large library, especially for educational/activity shots.

## Upload CLI Command

```bash
# Example: upload all event images at once using AWS CLI
aws s3 cp ./seed-images/events/ s3://skillspark-dev/events/ --recursive --content-type "image/jpeg"
aws s3 cp ./seed-images/orgs/   s3://skillspark-dev/orgs/   --recursive --content-type "image/jpeg"
aws s3 cp ./seed-images/profiles/ s3://skillspark-dev/profiles/ --recursive --content-type "image/jpeg"

# For LocalStack:
aws --endpoint-url=http://localhost:4566 s3 cp ./seed-images/ s3://skillspark-dev/ --recursive
```
